Java:

In 1991, James Gosling and his team at Sun Microsystems began working on a new programming language called Oak. In 1995, Oak was renamed Java and released to the public. Java quickly became popular for developing web applications, and it is now one of the most widely used programming languages in the world. Java is a general-purpose programming language that can be used to develop a wide variety of applications, including web applications, mobile apps, desktop apps, and enterprise software. Java is known for its portability, security, and performance. Java is also known for its large and active community of developers.
